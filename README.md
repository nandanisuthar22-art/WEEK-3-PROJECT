#week3: project
📊 STEM Education: Student Performance Analysis

A data analysis project focused on understanding student performance in STEM subjects (Math, Science, etc.) using Python. This project includes data cleaning, validation, and interactive visualizations to uncover insights from educational data.

🚀 Project Overview

This project analyzes student performance data to:

Clean and preprocess raw educational datasets
Validate and ensure data quality
Explore relationships between variables like:
Study hours 📚
Attendance 📅
Subject scores 📈
Create interactive visualizations for better insights
🧠 Key Features
✅ Data Cleaning (handling invalid values & formatting)
✅ Data Validation checks
📊 Interactive visualizations using Plotly
📈 Performance distribution analysis
🔍 Insightful relationships (e.g., study hours vs scores)
